# Quick Notes Extension

A premium, neobrutalist-style sticky notes Chrome extension built with React, TypeScript, and Vite.

## Features
- **Flexible Resizing**: Resize notes freely on the screen.
- **Paint & Text**: Write notes or draw directly on the sticky surface.
- **Smart Layout**: Automatic fluid layout that prevents horizontal scrolling.
- **Classic UI**: High-contrast, premium "Classic" design system.
- **Auto-save**: All notes and settings are persisted via Chrome Storage.

## Development
1. Install dependencies: `npm install`
2. Build for production: `npm run build`
3. Load the `dist/` folder into Chrome Extensions (`chrome://extensions`).

## Tech Stack
- React 18
- TypeScript
- Vite
- Lucide React Icons
